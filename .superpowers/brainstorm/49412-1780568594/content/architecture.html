Sistem Mimarisi

İki ayrı parça: veri toplama (tek seferlik) + frontend entegrasyonu

Genel Akış

1. VERİ TOPLAMA (bir kez çalıştır)
scripts/collect-services.py
→ Google Places API (18 kategori × İstanbul ilçeleri)
→ Deduplikasyon (Place ID'ye göre)
→ src/services-data.json (300–500 servis)
2. FRONTEND (mevcut App.jsx'e eklenir)
services-data.json bundle'a dahil edilir
Teşhis sonrası → "📍 Servis Bul" CTA
→ Tarayıcı geolocation API
→ Client-side Haversine filtre + sıralama
→ Servis listesi + 📞 Ara butonu
3. GÜNCELLEME (opsiyonel, periyodik)
Script yeniden çalıştır → JSON güncelle → git push → Vercel otomatik deploy
